## 🏛️ 宇宙架构

本宇宙由两大核心叙事构成一个辩证统一的整体：

- **《炽热堡垒》**: 半埋于岩浆湖边的绝对秩序堡垒，一个具备完整生产、防御与生活循环的终极社会工程学实验场。它是规则、结构与现实的**定义者**。
- **《轮回诗章》**: 在永恒重置的牢笼中，关于记忆、身份与爱的超维映射实验。它是情感、变量与可能的**歌颂者**。
